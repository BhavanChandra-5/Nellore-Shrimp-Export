ShrimpLogix - Nellore Shrimp Export Analytics

Project Overview:
This is a SQL-based project focused on the shrimp export industry in Nellore. I designed this database to track how shrimp batches move from local farmers in areas like 
Indukurpet and Kavali to international markets like the USA and Japan.

Database Tables:
Farmers: Details of local pond owners and their locations.
Harvests: Tracks batch weight and shrimp types (Vannamei and Tiger).
Export Orders: Shipping status and global pricing info.

What I implemented:
- Managed a dataset of 100 real-time records to simulate a live supply chain.
- Used SQL Joins to link farmers with their global sales and revenue.
- Applied Normalization to keep the data clean and avoid repetition.
- Wrote analytical queries to find out which Nellore region has the highest production.

How to use:
- Open the ShrimpLogix.sql file in MySQL Workbench.
- Run the full script to create tables and insert the 100 rows of data.
- Check the bottom of the script for sample queries to see the results.
